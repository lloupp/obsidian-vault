Meu vault pessoal para anotações e conhecimento.

├── 02 - Knowledge/

2. Configurações em `.obsidian/` (opcional)

## Sincronização

Repositório Git para versionamento e sincronização entre dispositivos.

Não versionar:
- `.trash/`
- `~temp/`
